# Description
A simple OOP Number to Roman-Numeral conversion library.

# Installation:

```
composer install
```


# Example:

see index.php

```
// Instantiate roman numeral converter
$o = new RomanNumeralConverter();

// Optional setting of roman numerals
$o->setRomanNumeral(1, 'Z');
$o->setRomanNumeral(5, 'P');
```
